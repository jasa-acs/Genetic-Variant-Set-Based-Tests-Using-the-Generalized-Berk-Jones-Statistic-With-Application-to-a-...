The online supplement files provided with this submission include:

1. ACC_Form_GBJ - Holds description of the data (GWAS from the CGEMS consortium) analyzed in this paper,
description of the code used to analyze the data, and instructions for reproducibility. Complies with
the ACC instructions given in the JASA Applications and Case Studies instructions to authors.

2. code_to_reproduce - Folder holding all the code describe in ACC_Form_GBJ. See the form for detailed
instructions on use.

3. GBJ_supplement_final - Online supplementary material including additional proofs, figures, and
tables referenced in main manuscript.
